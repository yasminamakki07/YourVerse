Description :
Your_Verse is a frontend web app built with React and styled using Chakra UI. It allows users to create, read, and manage their personal verses in a comforting interface. The app emphasizes ownership, privacy, and emotional safety, with features like owner-only edit/delete and a sanctuary-inspired design, focusing on the impact of a word .

Features :
Create, read, and manage personal verses
Owner-only edit/delete for privacy and agency
Sanctuary-inspired UI using Chakra components
Deployed on GitHub Pages

Technologies Used :
React (frontend only)
Chakra UI (CSS styling and components)
Git & GitHub for version control
GitHub Pages for deployment

Installation:
Run npm install
Run npm start

Build & Deploy:
Run npm run build
Run npm run deploy
