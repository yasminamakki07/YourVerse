YourVerse ✨
YourVerse is a gentle React‑based web application designed as a sanctuary for writing and keeping personal verses with privacy and agency. It highlights the beauty of words and how writing can change the world.
Live at: https://yasminamakki07.github.io/YourVerse/

Description
YourVerse is a frontend web app built with React and styled using Chakra UI. It allows users to create, read, and manage their personal verses in a comforting interface. The app emphasizes ownership, privacy, and emotional safety, with features like owner‑only edit/delete and a sanctuary‑inspired design. The project demonstrates how digital spaces can be shaped to feel safe, poetic, and empowering.

Features
Create, read, and manage personal verses
Owner‑only edit/delete for privacy and agency
Sanctuary‑inspired UI using Chakra components
Fully deployed frontend on GitHub Pages
Backend powered by Node.js  and Express, deployed on Render
Database integration with MySQL on Railway

Technologies Used
React (frontend)
Chakra UI (styling and components)
Framer Motion (animations)
Node.js  and Express (backend)
MySQL on Railway (database)
Git & GitHub for version control
GitHub Pages for frontend deployment
Render for backend deployment

Setup Instructions
Clone the repository from GitHub.
Install dependencies using npm install.
Run the app locally with npm start.
Build the app with npm run build.
Deploy to GitHub Pages with npm run deploy.
